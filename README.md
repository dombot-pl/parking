# parking
Parking for domains, show conatact after redirection based on IP number and CNAME

[parking.dombot.pl](https://parking.dombot.pl/)

## install on VPS

    git clone https://github.com/dombot-pl/parking.git
    
    sh install.sh
    
    sh status.sh
    
    sh start.sh
